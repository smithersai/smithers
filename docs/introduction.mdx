Smithers is a JSX-first framework for building deterministic, resumable AI workflows. You define the workflow as a TypeScript component tree, and Smithers handles execution ordering, persistence, approvals, retries, and observability.

## Start Here

- [JSX API](/jsx/overview) - Learn the component model and how execution works.
- [Installation](/jsx/installation) - Set up TypeScript, TSX, and optional MDX prompts.
- [Quickstart](/jsx/quickstart) - Build and run a two-step workflow.

## Shared Runtime

No matter how large the workflow gets, Smithers handles the same runtime concerns:

- durable execution and resumability
- retries, timeouts, and approval gates
- SQLite persistence
- parallel execution and loops
- structured events, metrics, and observability

The Core Concepts, Runtime, CLI, and Integrations sections describe the shared engine underneath that JSX authoring layer.
