Ticket IDs must be lowercase kebab-case slugs derived from the title (e.g. `add-auth-middleware`); never numeric IDs like `T-001` because they collide across discovery runs. Limit each discovery batch to 3-5 tickets so each batch benefits from prior implementation context.
