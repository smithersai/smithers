This material is now in [Recipes](/recipes#mdx-prompt-with-auto-injected-schema).

When a `<Task>` child is a JSX/MDX element and an output schema is set, Smithers auto-injects `props.schema` with a JSON example derived from the Zod schema — no manual passing required. If running tests, register `mdxPlugin()` in the `[test]` preload as well.
