This material is now in [Recipes](/recipes).

For 5+ tasks, use a three-file pattern per step: `Component.schema.ts` (Zod), `Component.mdx` (prompt with auto-injected `{props.schema}`), and `Component.tsx` (Task wiring). Keep `workflow.tsx` as pure composition; centralize `createSmithers()` in `smithers.ts`, agents in `agents.ts`, and shared knobs (timeouts, max iterations) in `config.ts`.
