This material is now in [CLI Overview](/cli/overview).

Per-run NDJSON event log lives at `.smithers/executions/<runId>/logs/stream.ndjson` (one `SmithersEvent` per line). Useful inspection commands: `inspect`, `logs --follow`, `chat`, `why`, `node`, `ps`, `graph`. Most internal state is queryable via SQLite tables prefixed `_smithers_*`.
