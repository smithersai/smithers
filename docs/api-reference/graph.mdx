## `build_graph`

Build an execution graph from a target workflow.

```python
from smithers import build_graph

graph = build_graph(deploy_workflow)
```

### Signature

```python
def build_graph(target: Workflow) -> WorkflowGraph
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `target` | `Workflow` | The target workflow to build the graph for |

### Returns

A `WorkflowGraph` containing all workflows and their dependencies.

### Raises

| Exception | Description |
|-----------|-------------|
| `ValueError` | Missing dependency or circular dependency detected |

### Example

```python
@workflow
async def step1() -> A:
    ...

@workflow
async def step2(a: A) -> B:
    ...

graph = build_graph(step2)
# Graph includes both step1 and step2
```

## `run_graph`

Execute a workflow graph.

```python
from smithers import run_graph

result = await run_graph(graph)
```

### Signature

```python
async def run_graph(
    graph: WorkflowGraph,
    cache: Cache | None = None,
    max_concurrency: int | None = None,
    timeout: int | None = None,
    node_timeout: int | None = None,
    retry_policy: RetryPolicy | None = None,
    fail_fast: bool = False,
    on_progress: Callable[[WorkflowEvent], Awaitable[None]] | None = None,
    dry_run: bool = False,
    return_all: bool = False,
) -> Any
```

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `graph` | `WorkflowGraph` | required | The graph to execute |
| `cache` | `Cache \| None` | `None` | Cache for skipping unchanged workflows |
| `max_concurrency` | `int \| None` | `None` | Max parallel workflows |
| `timeout` | `int \| None` | `None` | Global timeout in seconds |
| `node_timeout` | `int \| None` | `None` | Default per-node timeout |
| `retry_policy` | `RetryPolicy \| None` | `None` | Retry policy for failed nodes |
| `fail_fast` | `bool` | `False` | Stop all execution on first failure |
| `on_progress` | `Callable` | `None` | Callback for progress events |
| `dry_run` | `bool` | `False` | Preview execution without running |
| `return_all` | `bool` | `False` | Return all outputs, not just root |

### Returns

The output of the root (target) workflow. If `return_all=True`, returns an `ExecutionResult` with all outputs.

### Example

```python
from smithers import SqliteCache, RetryPolicy

cache = SqliteCache("./cache.db")
result = await run_graph(
    graph,
    cache=cache,
    max_concurrency=4,
    timeout=300,
    node_timeout=60,
    retry_policy=RetryPolicy(max_attempts=3),
)
```

## `run_graph_with_store`

Execute a workflow graph with a persistent SQLite store.

```python
from smithers import run_graph_with_store

result = await run_graph_with_store(
    graph,
    store_path="./smithers.db",
    timeout=300,
)
```

This variant persists all events, runs, and node states to SQLite for later inspection.

## `WorkflowGraph`

The graph data structure.

```python
@dataclass
class WorkflowGraph:
    root: str
    nodes: dict[str, WorkflowNode]
    edges: list[tuple[str, str]]
    levels: list[list[str]]
```

### Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| `root` | `str` | Name of the target workflow |
| `nodes` | `dict[str, WorkflowNode]` | All workflows in the graph |
| `edges` | `list[tuple[str, str]]` | Dependency edges (from, to) |
| `levels` | `list[list[str]]` | Parallelization groups |

### Methods

#### `mermaid`

Generate a Mermaid diagram.

```python
print(graph.mermaid())
# graph LR
#     step1 --> step2
#     step2 --> step3
```

## `WorkflowNode`

A node in the workflow graph.

```python
@dataclass
class WorkflowNode:
    name: str
    output_type: type[BaseModel]
    dependencies: list[str]
    requires_approval: bool
    approval_message: str | None
```

### Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Workflow function name |
| `output_type` | `type[BaseModel]` | What the workflow produces |
| `dependencies` | `list[str]` | Names of upstream workflows |
| `requires_approval` | `bool` | Whether approval is needed |
| `approval_message` | `str \| None` | Approval prompt |

## `WorkflowResult`

Result of a single workflow execution.

```python
@dataclass
class WorkflowResult:
    name: str
    output: Any
    cached: bool
    duration_ms: float
```

### Attributes

| Attribute | Type | Description |
|-----------|------|-------------|
| `name` | `str` | Workflow name |
| `output` | `Any` | The workflow's output |
| `cached` | `bool` | Whether result was from cache |
| `duration_ms` | `float` | Execution time in milliseconds |
