---
title: Installation
description: Install Smithers and its core dependencies.
---

**Requirements**

- Bun >= 1.3 (Smithers uses Bun SQLite and Bun runtime APIs)
- TypeScript >= 5 (peer dependency)

**Install**

```bash
bun add smithers ai @ai-sdk/anthropic drizzle-orm drizzle-zod
```

**Optional: JJ (Jujutsu) for revert support**

```bash
brew install jj
```

If JJ is not installed, revert functionality is unavailable and JJ pointers are recorded as `null`.
