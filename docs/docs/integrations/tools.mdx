---
title: Built-in Tools
description: File and shell tools for agent tasks.
---

Smithers provides a small toolset compatible with the AI SDK.

```ts
import { read, write, edit, grep, bash, tools } from "smithers/tools";
import { agent } from "ai";

const codeAgent = agent({
  model: anthropic("claude-sonnet-4-20250514"),
  tools: { read, grep, bash },
});
```

**Available tools**

- `read`: Read a file
- `write`: Write a file
- `edit`: Apply a unified diff patch
- `grep`: Search for a pattern in files
- `bash`: Execute a shell command (network disabled unless `allowNetwork` is true)

**Tool sandbox**

Tools are sandboxed to `rootDir` (default: workflow directory). Use `allowNetwork` in `runWorkflow` or the CLI to enable network access for `bash`.
