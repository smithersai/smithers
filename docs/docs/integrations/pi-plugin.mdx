---
title: PI Plugin Client
description: Call a Smithers server from another process.
---

The `smithers/pi-plugin` entrypoint provides a small HTTP client for the Smithers server.

```ts
import { runWorkflow, approve, deny, streamEvents } from "smithers/pi-plugin";

const run = await runWorkflow({
  workflowPath: "./workflow.tsx",
  input: { description: "Fix bugs" },
  baseUrl: "http://127.0.0.1:7331",
});

for await (const event of streamEvents({ runId: run.runId })) {
  console.log(event.type);
}
```

**Functions**

- `runWorkflow({ workflowPath, input, runId?, baseUrl?, apiKey? })`
- `resume({ workflowPath, runId, baseUrl?, apiKey? })`
- `approve({ runId, nodeId, iteration?, note?, baseUrl?, apiKey? })`
- `deny({ runId, nodeId, iteration?, note?, baseUrl?, apiKey? })`
- `streamEvents({ runId, baseUrl?, apiKey? })`
- `getStatus({ runId, baseUrl?, apiKey? })`
- `getFrames({ runId, tail?, baseUrl?, apiKey? })`
- `cancel({ runId, baseUrl?, apiKey? })`
- `listRuns({ limit?, status?, baseUrl?, apiKey? })`
