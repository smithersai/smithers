---
title: renderFrame
description: Render a workflow to a graph snapshot.
---

`renderFrame(...)` renders the workflow tree without executing tasks. It returns a `GraphSnapshot` with an XML representation and task descriptors.

```ts
import { renderFrame } from "smithers";
import workflow from "./workflow";

const snapshot = await renderFrame(workflow, {
  runId: "preview",
  iteration: 0,
  input: { description: "Fix bugs" },
  outputs: {},
});

console.log(snapshot.frameNo, snapshot.tasks.length);
```

**GraphSnapshot**

- `runId`: run id
- `frameNo`: frame number
- `xml`: XML tree (or null)
- `tasks`: array of task descriptors
