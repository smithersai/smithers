---
title: Best Practices
description: Guidelines for effective Smithers workflows.
---

**Give agents big, coherent tasks**

Agents perform better with a single, well-scoped task than with many micro-steps. Only split tasks when context changes significantly or you need explicit gating.

**Use measurable stop conditions**

Loops should stop based on a concrete signal (tests passing, approval flag, score threshold).

**Ask for validation explicitly**

If you need tests or verification, put it in the prompt. Don’t assume the model will run checks without being asked.

**Request structured reports**

Ask for outputs you want to inspect later (files changed, tests run, metrics). These should map to your output table schema.

## Example: Review loop with Ralph

```tsx
// workflow.tsx
import { smithers, Workflow, Task, Sequence, Ralph } from "smithers";
import { agent } from "ai";
import { anthropic } from "@ai-sdk/anthropic";
import { drizzle } from "drizzle-orm/bun-sqlite";
import { sqliteTable, text, integer, primaryKey } from "drizzle-orm/sqlite-core";

const inputTable = sqliteTable("input", {
  runId: text("run_id").primaryKey(),
  description: text("description").notNull(),
});

const reviewTable = sqliteTable(
  "review",
  {
    runId: text("run_id").notNull(),
    nodeId: text("node_id").notNull(),
    iteration: integer("iteration").notNull().default(0),
    approved: integer("approved", { mode: "boolean" }).notNull(),
    feedback: text("feedback").notNull(),
  },
  (t) => ({
    pk: primaryKey({ columns: [t.runId, t.nodeId, t.iteration] }),
  }),
);

const outputTable = sqliteTable(
  "output",
  {
    runId: text("run_id").notNull(),
    nodeId: text("node_id").notNull(),
    finalSummary: text("final_summary").notNull(),
  },
  (t) => ({
    pk: primaryKey({ columns: [t.runId, t.nodeId] }),
  }),
);

const schema = { input: inputTable, output: outputTable, review: reviewTable };
const db = drizzle("./workflow.db", { schema });

const reviewer = agent({
  model: anthropic("claude-sonnet-4-20250514"),
  system: "You are a strict reviewer. Return JSON with approved and feedback.",
});

export default smithers(db, (ctx) => (
  <Workflow name="review-loop">
    <Ralph
      until={ctx.output(schema.review, { nodeId: "review" }).approved}
      maxIterations={3}
      onMaxReached="return-last"
    >
      <Sequence>
        <Task id="review" output={schema.review} agent={reviewer}>
          {`Review: ${ctx.input.description}. Return JSON with approved (boolean) and feedback (string).`}
        </Task>
      </Sequence>
    </Ralph>

    <Task id="summary" output={schema.output}>
      {{
        finalSummary: ctx.output(schema.review, { nodeId: "review" }).approved
          ? "Approved"
          : "Not approved",
      }}
    </Task>
  </Workflow>
));
```
