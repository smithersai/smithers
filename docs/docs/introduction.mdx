title: Introduction
description: Define AI workflows with React + SQLite, executed deterministically.
---

Smithers is a React-based workflow runtime for AI tasks. You define a workflow graph with JSX, store inputs/outputs in SQLite (via Drizzle), and execute deterministically with durable state.

**What you get**

- Deterministic execution with resumable runs
- Durable outputs in SQLite (Drizzle)
- Human approvals and retries
- Built-in tool sandbox for agent tasks
- Event stream for real-time progress

Smithers is a library. It does not depend on the desktop app and can be used from scripts, services, or CI.
